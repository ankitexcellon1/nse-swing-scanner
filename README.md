# NSE Swing Scanner

This repository runs an automated NSE swing-trading scanner.

Current version:
- Beginner MVP
- Technical scan only
- No auto-trading
- No guaranteed buy/sell recommendation
- Report generated in Markdown format

Planned workflow:
1. Fetch market data
2. Calculate indicators
3. Apply swing-trading filters
4. Generate daily report
5. Review output manually before any trading decision

Disclaimer:
This is for research and learning only. Final trading decisions, position sizing, and risk management remain the user's responsibility.
